# Boolean-Expression-Parser
Цей репозиторій містить консольну програму мовою C++ для синтаксичного аналізу, обробки та автоматичного спрощення логічних (булевих) виразів. Програма приймає текстові формули, будує на їх основі абстрактне синтаксичне дерево (AST) та застосовує закони алгебри логіки для мінімізації виразу без зміни його кінцевого результату.

Функціональні можливості програми включають посимвольний парсинг виразів, які складаються зі змінних (латинські літери), логічних констант (0 та 1), круглих дужок та базових операторів: кон'юнкції (&), диз'юнкції (|) та заперечення (!). Після побудови дерева програма автоматично виконує первинне спрощення, використовуючи математичні правила, такі як закони ідемпотентності, поглинання, виключеного третього та операції з нулем і одиницею.

Також передбачено можливість інтерактивної підстановки значень. Програма знаходить усі змінні у введеному виразі, запитує для них числові значення (0 або 1) у користувача, після чого виконує підстановку, переобчислення та зводить вираз до фінального результату.
